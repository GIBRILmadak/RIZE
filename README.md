# RIZE 🚀

RIZE is a web platform designed for ambitious creators and builders who want to **document their projects, track their progress daily, and gain visibility over time**.

Instead of only showcasing finished products, RIZE focuses on the *process*: effort, consistency, evolution, and learning.

## 🎯 Why RIZE exists
Most platforms highlight results.  
RIZE highlights **trajectory**.

Creators can:
- Create projects ("Arcs")
- Document daily progress ("Traces")
- Build credibility through consistency
- Inspire others by showing real work, not just outcomes

## 🧠 Who it's for
- Indie hackers
- Students & early founders
- Developers, designers, creators
- Anyone building something over time

## 🛠 Tech stack
- Vanilla JavaScript
- HTML / CSS
- Supabase (auth + database)
- Netlify (deployment)

## 🚧 Project status
RIZE is in **active early development**.
Core features exist, but UX, structure, and scalability are still being refined.

This is **not** a finished product — feedback and collaboration are welcome.
